# AttestationStation

Map of attestations
```solidity
mapping(address => mapping(address => mapping(bytes32 => bytes))) public attestations;
```

Attestation structure:
```solidity
struct AttestationData {
    address about;
    bytes32 key;
    bytes val;
}
```

The structure of `bytes val;` are described in [Attestations](../1_attestations.md).
